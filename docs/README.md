# Clack Docs

This folder holds project documentation that is more detailed than the top-level README.

## Documents

- [Privacy model](PRIVACY_MODEL.md): what Clack stores, what stays local, and which privacy areas need review before stable releases.
- [Release process](RELEASE_PROCESS.md): versioning, packaging, GitHub Releases, Homebrew, signing, and notarization notes.
- [Roadmap](ROADMAP.md): current alpha scope, next UI work, distribution plans, and later ideas.

## Maintenance

Update these docs in the same pull request as the behavior they describe. Privacy, release, storage, and packaging changes should not land with stale documentation.
