# Support

Clack is an early open-source project. The best place to get help is GitHub Issues.

## Use Issues For

- Bug reports.
- Feature requests.
- UX feedback.
- Documentation problems.
- Questions about expected behavior.

## Before Opening an Issue

- Search existing issues first.
- Remove private clipboard contents from examples.
- Include your macOS version when reporting app behavior.
- Include your Clack version and install method.
- Include screenshots or recordings for visual bugs.
- Mention whether the issue happens after restarting Clack.

## Useful Details for Bug Reports

Please include anything that helps reproduce the issue without exposing private data:

- The kind of clipboard item involved: text, rich text, file, image, or Universal Clipboard.
- Whether the item was copied from a specific app.
- Whether the item appears in the menu, preview card, preferences, or stored history.
- Any preference setting that seems related.

## Security Reports

Do not use public issues for security or privacy vulnerabilities. See [SECURITY.md](SECURITY.md).
